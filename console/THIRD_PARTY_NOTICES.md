# Third-party notices

The console is distributed under the repository's Apache-2.0 license. Direct
JavaScript dependencies are pinned by `package-lock.json` and declare the
following licenses:

| Dependency | Version | Declared license |
|---|---:|---|
| express | 5.1.0 | MIT |
| helmet | 8.1.0 | MIT |
| keycloak-js | 26.2.0 | Apache-2.0 |
| react / react-dom | 19.1.0 | MIT |
| TypeScript | 5.8.3 | Apache-2.0 |
| Vite | 7.3.6 | MIT |
| @vitejs/plugin-react | 4.6.0 | MIT |
| concurrently | 9.2.0 | MIT |
| @types/react / @types/react-dom | 19.1.x | MIT |

The lockfile's transitive package metadata contains only MIT, Apache-2.0, ISC,
BSD-3-Clause, 0BSD, and CC-BY-4.0 declarations; no package lacks a declared
license in the lockfile. The CC-BY-4.0 entry is `caniuse-lite`, which contains
browser support data derived from [Can I Use](https://caniuse.com/) and is used
only by the build toolchain. Its attribution and license must be preserved when
redistributing that data.

This inventory records package metadata and does not replace the organization's
final dependency, export-control, vulnerability, or image compliance review.
