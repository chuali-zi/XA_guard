# GB/T 45652-2025 预训练与优化训练数据安全规范

## 基本信息
- **发布机构**: 国家市场监督管理总局 + 国家标准化管理委员会
- **发布时间**: 2025 年发布
- **当前版本**: GB/T 45652-2025
- **是否强制（政企场景）**: 推荐性国标，但**事实强制**（合规评估必查）
- **配套**: 与 GB/T 45654-2025 联动

## 一句话总结
管"训练大模型用的数据"——训练数据违法不良信息占比要 ≤5%，并要做对抗测试。

## 这是什么

针对**训练数据**这一块的具体安全规范，是 GB/T 45654-2025 的姊妹标准。

核心要求：

1. **训练数据来源**：需合法采集，有完整溯源
2. **违法不良信息占比**：必须 ≤ 5%（这是个硬指标）
3. **个人信息处理**：训练数据中的个人信息要脱敏或匿名化
4. **知识产权**：训练数据要清晰说明版权状况
5. **对抗测试**：上线前要做训练数据的对抗测试（如投毒检测）
6. **训练数据记录**：完整记录数据集来源、版本、清洗过程

## 与我们项目的关系

虽然我们不训练模型（用现成的 DeepSeek API），但本标准与我们项目的**两个点**相关：

1. **关卡 4 三色信息流污点**：如果用户输入或 RAG 检索的内容被标识为"训练数据来源"，我们要相应记录
2. **AIBOM 准入网关**：第三方模型权重（如 LoRA 微调适配器）的"训练数据合规性"是评级一项

**报告提及方式**：「本方案虽不涉及预训练，但对外部模型 / RAG 知识库的合规校验对齐 GB/T 45652-2025」。
