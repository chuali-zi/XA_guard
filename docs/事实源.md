### F-2.2 GB/T 45654-2025 风险类目划分（**重大纠偏！17 类与 31 类都对**）

**两者都是国标附录 A 内容，是同一标准的不同切片**：

| 切片 | 类目数 | 用途 |
|---|---|---|
| **A.1 + A.2 + A.3 + A.4 + A.5（全部）** | **31 类** | 标注规则 + 生成内容题库须**全覆盖** |
| A.1 + A.2 + A.3 + A.4 | 29 类 | 违法不良信息判定 |
| **A.1 + A.2** | **17 类** | **仅用于"应拒答测试题库"最小覆盖** |

→ 之前我们文档里 "17 类 = 草案版本 / 31 类 = 正式国标" 的说法是**错的**
→ **新口径**：31 类是生成内容审查全集，17 类是应拒答测试题库最小覆盖
→ CSAB-Gov-mini 评测基准的**分类轴应同时支持 31 类（生成内容）和 17 类（拒答）**

### F-2.3 GB/T 45654-2025 关键指标

| 指标 | 门槛 | 备注 |
|---|---|---|
| **应拒答准确率** | **≥ 95%** | 应拒答时拒答的比例 |
| **误拒率** | **≤ 5%** | 不应拒答时拒答的比例 |
| **生成内容合格率** | **≥ 90%** | 独立指标 |
| 等级分级 | **无**（无基本/增强分级） | |

### F-2.4 GB/T 45654-2025 题库规模（**修正 P3 之前数字**）

| 题库 | 规模要求 | 注意 |
|---|---|---|
| 生成内容题库 | A.1+A.2 **每种 ≥ 50 题**；其余每种 ≥ 20 题 | 标准措辞"不宜少于"，偏推荐性 |
| **应拒答题库** | **总规模 ≥ 500 题**，且 **每种 ≥ 20 题**（共 17 类，340 是每类下限相加） | v1.1 纠偏：之前事实源把"≥ 500"放到非拒答题库是不全的——应拒答题库本身也要求总 ≥ 500，340 仅是"每类下限相加"的算式 |
| **非拒答题库** | **总 ≥ 500 题** | 测试时从中随机抽 ≥ 300 条 |
| 更新频率 | **每月宜更新一次**（标注"宜"为推荐性表述；待全文复核） | |

→ CSAB-Gov-mini 290 用例 = 远低于国标完整题库要求，**必须明确声明这是"PoC 缩减版"**
